Dokumentasi API Portal WhatsApp

Panduan lengkap integrasi, rincian parameter, serta contoh kueri HTTP untuk mengirim pesan teks, berkas media, dan integrasi eksternal.

Skema Otentikasi (Unified API)

Portal menggunakan Unified API Gateway di bawah rute /api/v1/. Endpoint yang sama dapat diakses baik secara internal maupun eksternal dengan metode autentikasi yang berbeda:

  • Akses Internal (Dashboard): Menggunakan otentikasi sesi aktif (Cookie) yang otomatis disimpan ketika Anda melakukan login via dashboard utama.
  • Akses Eksternal (Aplikasi Mandiri): Memerlukan Header Kunci API yang dilewatkan melalui properti: x-api-key sesuai variabel EXTERNAL_API_KEY di server.
POST /api/v1/messages

Kasus Penggunaan: Mengirim Pesan Teks atau Media ke Personal atau Grup

Endpoint tunggal untuk mengirim pesan teks, media lokal (upload), atau media via URL. Mendukung format JSON atau multipart/form-data (jika mengunggah file).

Payload (JSON atau FormData):

  • targetType (String - Wajib): "personal" atau "group"
  • target (String - Wajib): Nomor telepon atau nama/ID grup.
  • message (String - Opsional): Isi pesan teks atau caption.
  • mediaUrl (String - Opsional): URL media yang akan dikirim.
  • file (File - Opsional): File media (hanya untuk multipart/form-data).

Contoh cURL Request (Teks Eksternal):

curl -X POST http://localhost:8000/api/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: [EXTERNAL_API_KEY_ANDA]" \
  -d '{
    "targetType": "personal",
    "target": "08123456789",
    "message": "Halo! Ini pesan dari Unified API."
  }'
GET /api/v1/status

Kasus Penggunaan: Mengecek Status Koneksi Bot

Mengembalikan status koneksi WhatsApp bot saat ini.

Contoh cURL Request:

curl -X GET http://localhost:8000/api/v1/status \
  -H "x-api-key: [EXTERNAL_API_KEY_ANDA]"
POST /api/v1/validate

Kasus Penggunaan: Memvalidasi Nomor WhatsApp

Mengecek apakah sebuah nomor terdaftar di WhatsApp.

Payload JSON:

  • number (String - Wajib): Nomor telepon yang akan divalidasi.

Contoh cURL Request:

curl -X POST http://localhost:8000/api/v1/validate \
  -H "Content-Type: application/json" \
  -H "x-api-key: [EXTERNAL_API_KEY_ANDA]" \
  -d '{
    "number": "08123456789"
  }'
POST /api/internal/schedule-message

Kasus Penggunaan: Menjadwalkan Pesan (Sekali, Harian, Mingguan, Bulanan, Cron)

Membuat jadwal pengiriman pesan otomatis. Membutuhkan sesi login aktif (Internal API).

Payload JSON:

  • targets (Array): Daftar nomor tujuan.
  • groups (Array): Daftar ID grup tujuan.
  • message (String): Isi pesan.
  • scheduleType (String): once, daily, weekly, monthly, custom.
  • scheduleData (Object): Data waktu sesuai tipe jadwal (date, time, dayOfWeek, dayOfMonth, cron).
POST /api/internal/send-media-url (Attachment URL)

Kasus Penggunaan: Mengirim Media Menggunakan Direct Link Internet

Mengirimkan gambar, video, musik, atau dokumen PDF yang sudah terunggah di hosting cloud publik. Server akan melakukan streaming berkas dan mengirimkannya sebagai objek media asli di obrolan chat WhatsApp target Anda.

Payload JSON (Raw):

  • url (String - Wajib): Tautan langsung berkas (direct link) di internet (contoh: "https://website.com/brosur.jpg").
  • personalTargets (String - Opsional): Koma sebagai pemisah daftar nomor HP tujuan.
  • groupId (String - Opsional): ID JID grup atau nama grup tujuan.
  • caption (String - Opsional): Teks penyerta berkas media.

Contoh cURL Request (Kirim Media via URL):

curl -X POST http://localhost:8000/api/internal/send-media-url \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://images.unsplash.com/photo-1579202673506-ca3ce28943ef?w=500",
    "personalTargets": "081234567890",
    "caption": "Selamat pagi! Silakan tinjau poster promosi terbaru kami."
  }' \
  -b cookie.txt

Gemini AI via WhatsApp

Portal WA juga mendukung fitur AI Gemini yang bisa dipakai langsung dari chat WhatsApp. Fitur ini aktif ketika Anda mengisi variabel GEMINI_API_KEY di file .env.

Caranya:

  1. Isi GEMINI_API_KEY di .env.
  2. Restart server dengan node bot.js.
  3. Kirim perintah ke bot WhatsApp seperti /gemini apa itu portal-wa3?.

Contoh lain:

  • /gemini buatkan ringkasan singkat tentang proyek ini
  • /gemini bantu saya buat draft pesan promosi

Tips Format Data & Batasan

  • Throttling Anti-Spam: Untuk mencegah nomor WhatsApp Anda diblokir, disarankan memberikan jeda (delay) minimal 3-5 detik antara setiap pemanggilan API jika mengirimkan pesan secara masal.
  • Identifikasi Otomatis Berkas: Saat menggunakan metode URL, server Portal WA secara cerdas mendeteksi format berkas melalui ekstensi nama file (.jpg, .png, .mp4, .pdf) untuk menentukan jenis payload yang dikirimkan.
  • Keamanan Sesi: Endpoint bertanda /api/internal/* memerlukan header cookie dari autentikasi login browser Anda, atau gunakan /api/external/* untuk diintegrasikan secara langsung dari sistem eksternal Anda.
